# Portfólio C 🖥️

Exercícios em C desenvolvidos durante o curso de programação.

## 📁 Exercícios

| Arquivo | Descrição |
|---|---|
| `idade-em-dias.c` | Converte idade em anos, meses e dias para total em dias |
| `area-retangulo.c` | Calcula a área de um retângulo |
| `verificador-voto.c` | Verifica se uma pessoa pode ou deve votar |
| `classificador-numero.c` | Classifica um número (positivo/negativo, par/ímpar, inteiro/decimal) |
| `classificador-triangulo.c` | Classifica um triângulo (equilátero, isósceles ou escaleno) |
| `ordenacao-tres-valores.c` | Ordena três valores em ordem crescente |
| `calculadora-media-notas.c` | Calcula média ponderada e verifica aprovação |
| `percentual-votos-eleicao.c` | Calcula percentual de votos brancos, nulos e válidos |
| `verificador-aposentadoria.c` | Verifica se uma pessoa pode se aposentar |
| `soma-pares.c` | Soma todos os números pares de 1 até N |
| `calculadora-imposto-renda.c` | Calcula imposto de renda com alíquota efetiva |
| `calculadora-pena-criminal.c` | Calcula pena ajustada com base em atenuantes e agravantes |
